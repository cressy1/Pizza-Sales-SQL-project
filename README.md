# Pizza Sales Analysis Using PostgreSQL

## Project Overview

This project presents an end-to-end sales analysis of a pizza restaurant dataset using **PostgreSQL** to uncover key business insights and evaluate overall sales performance. The analysis transforms raw transactional data into meaningful metrics that help understand customer purchasing behavior, product performance, and revenue trends.

The project began with importing the dataset into PostgreSQL, followed by data preparation tasks such as data type conversion, validation, and cleaning to ensure accurate analysis. SQL queries were then developed to calculate essential business KPIs, analyze sales trends over time, identify customer ordering patterns, and evaluate the performance of different pizza categories, sizes, and menu items.

By leveraging advanced SQL techniques, the project demonstrates how transactional sales data can be transformed into actionable insights that support strategic business decisions related to inventory management, menu optimization, pricing strategies, and operational planning.

---

## Project Objectives

* Evaluate overall sales performance using key business metrics.
* Analyze customer purchasing and ordering behavior.
* Identify top-selling and underperforming pizzas.
* Determine revenue contribution by pizza category and size.
* Discover peak business hours and weekly sales patterns.
* Measure order volume and customer purchasing trends.
* Generate actionable insights to improve sales and operational efficiency.
